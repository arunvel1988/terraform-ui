What is the Azure Provider?

The Azure Provider is the official Terraform plugin used to manage resources on Microsoft Azure. It enables provisioning and management of a broad set of Azure services like Virtual Machines, Storage Accounts, AKS, VNets, and more.

Use Cases:

  • Automate infrastructure provisioning on Azure
  • Define and manage cloud architecture as code
  • Secure and govern Azure resources programmatically

üìò What is a Workspace?

A Terraform Workspace is a way to manage separate state files for the same configuration. This is useful for managing different environments like dev, stage, and prod without duplicating code.

‚öôÔ∏è Common Workspace Commands

terraform workspace list           # Show available workspaces
terraform workspace new dev        # Create a new workspace named 'dev'
terraform workspace select dev     # Switch to 'dev' workspace
terraform workspace delete dev     # Delete the 'dev' workspace

üß† Usage Example in Code

You can dynamically use the workspace name inside your Terraform code like this: 

resource "aws_s3_bucket" "example" {
  bucket = "my-bucket-${terraform.workspace}"
}

This will name the bucket differently for each workspace (e.g., my-bucket-dev, my-bucket-prod).

What is LocalStack?

LocalStack is a fully functional local AWS cloud stack. It allows you to develop and test cloud applications entirely on your local machine, without ever needing to deploy to the real AWS cloud.

Use Cases:

  • Local testing of AWS infrastructure
  • CI/CD pipelines without cloud cost
  • Safe sandbox for experiments

Supported Services:

  • S3, Lambda, SNS, SQS, DynamoDB, API Gateway, and many more...

🌍 Terraform Overview

Terraform is an open-source Infrastructure as Code (IaC) tool developed by HashiCorp. It lets you define, preview, and deploy cloud and on-prem infrastructure using declarative configuration files.

  • ✅ Declarative Syntax: Define your infrastructure in .tf files, and Terraform figures out the rest.
  • 🚀 Providers: Integrate with AWS, Azure, GCP, Kubernetes, and many more using plugins.
  • 📦 State Management: Keeps track of infrastructure changes in a local or remote terraform.tfstate file.
  • 🔁 Plan & Apply Cycle: See a preview of changes with terraform plan before applying them with terraform apply.
